# Voice of Customer AI Agent

This project builds an autonomous AI agent that analyzes product reviews
and generates actionable insights for product, marketing, and support teams.

## Features

- Review ingestion pipeline
- Sentiment analysis
- Theme detection
- SQLite database
- Global analysis report
- Weekly delta report
- Conversational query agent
- Automated scheduler

## Tech Stack

Python  
SQLite  
NLP  
Scheduler  

## Run Project
python main.py

Reports will be generated in:

reports/global_report.md  
reports/weekly_report.md
